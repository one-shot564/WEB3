Файл Index.cshtml:

@model IEnumerable<HotelBooking.Models.Booking>: Это указание на тип модели, которая будет передана в представление — в данном случае, перечисление (IEnumerable) бронирований (Booking). Это означает, что в представление можно передать коллекцию объектов Booking.

<h2>Bookings</h2>: Заголовок страницы, который будет отображаться в браузере.

<p><a href="@Url.Action("Create", "Bookings")">Create New Booking</a></p>: Ссылка, которая позволяет пользователю перейти к форме создания нового бронирования. Это делается путём генерации URL, который вызовет метод Create в контроллере Bookings.

<table class="table">...</table>: Элемент HTML-таблицы с классом Bootstrap table для стилизации. В таблице есть заголовки столбцов и строки, генерируемые для каждого объекта бронирования в переданной модели.

@foreach (var booking in Model) { ... }: Цикл foreach, который перебирает все объекты бронирования, переданные в модели, и для каждого из них создаёт строку в таблице (<tr>).

Внутри цикла foreach, @booking.Id, @booking.RoomId, @booking.StartDate.ToString("d"), и @booking.EndDate.ToString("d") используются для отображения соответствующих свойств каждого объекта бронирования.
